🚀 INSTRUCCIONES RÁPIDAS PARA WINDOWS 🚀

¿Tienes problemas con Python? ¡Sigue estos pasos!

1️⃣ INSTALAR PYTHON:
   - Haz doble clic en: instalar_python.bat
   - O ve a: https://python.org/downloads
   - IMPORTANTE: Marca "Add Python to PATH" ✓

2️⃣ DESPUÉS DE INSTALAR PYTHON:
   - Reinicia la terminal/PowerShell
   - Ejecuta: python --version
   - Ejecuta: pip --version

3️⃣ INSTALAR DEPENDENCIAS:
   - Ejecuta: python install.py

4️⃣ EJECUTAR LA APLICACIÓN:
   - Ejecuta: python youtube_360_downloader.py

❓ PROBLEMAS COMUNES:
   - "pip no se reconoce": Python no está en PATH
   - "Python no se reconoce": Python no está instalado
   - Solución: Ejecuta instalar_python.bat

📚 MÁS AYUDA:
   - Lee README.md para instrucciones completas
   - Ejecuta instalar_python.bat para ayuda automática
